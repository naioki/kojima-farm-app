# 設定ファイル

このディレクトリには、店舗名と品目名の設定ファイルが保存されます。

## ファイル

- `stores.json`: 店舗名リスト
- `items.json`: 品目名正規化マップ

## 自動生成

これらのファイルは、アプリケーション初回起動時に自動生成されます。

## 手動編集

必要に応じて、これらのファイルを直接編集することもできます。

### stores.json の形式

```json
{
  "stores": [
    "鎌ケ谷",
    "五香",
    "八柱"
  ]
}
```

### items.json の形式

```json
{
  "青梗菜": ["青梗菜", "チンゲン菜", "ちんげん菜"],
  "胡瓜": ["胡瓜", "きゅうり", "キュウリ"],
  "長ネギ": ["長ネギ", "ネギ", "ねぎ"]
}
```
